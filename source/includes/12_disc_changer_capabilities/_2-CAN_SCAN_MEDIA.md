## can\_scan\_media

Disc Changer is capable of scanning media and returning disc information via the driver.

### Signature

`<can_scan_media></can_scan_media>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Returns

`None`


### Example

```xml
<capabilities>
   <can_scan_media>true</can_scan_media>
</capabilities>
```
